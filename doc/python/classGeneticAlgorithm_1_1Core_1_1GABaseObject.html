Detailed Description

The base object provides the functions repr and str.

The GABaseObject provides a custom behavior for GA objects, which allows for a useful repr() and str() implementation. Because every object on this library is derived from this class, the whole scope of the library can be appreciated in this class' inheritance diagram.
