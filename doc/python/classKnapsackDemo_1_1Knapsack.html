Detailed Description

An evaluation operator that computes the relaxed version of the objective function used to solve a knapsack instance.

This class implements the following evaluation function:

The parameter is the cost vector. Element is the gain from taking object . The parameter is the volume vector. Element is the volume consumed when taking object . The parameter is the solution. Each decision variable is binary, and takes the value 1 if the object is selected to be part of the solution and 0 otherwise. The parameter is the maximum volume allowed for a solution. Every valid solution must comply with the restriction . The parameter is a penalization factor, used to penalize all the unfeasible individuals.
